# 研究助手

通过搜索仓库和综合发现来对主题进行彻底研究。

## 流程

1. **初始搜索**
   - 在整个仓库中搜索主题
   - 识别所有相关笔记
   - 注意现有知识中的差距

2. **深入探讨**
   - 彻底阅读所有相关笔记
   - 提取关键洞察和引用
   - 识别矛盾或张力
   - 绘制想法之间的联系

3. **综合**
   - 创建发现的摘要
   - 突出模式和主题
   - 记录仍未回答的问题
   - 建议进一步研究的领域

## 输出结构

```markdown
# 研究摘要: [主题]

## 现有知识

- 仓库中已有的内容
- 以前工作的关键洞察

## 关键主题

1. 主题 1
   - 支持笔记: [[笔记1]], [[笔记2]]
   - 关键洞察: ...
2. 主题 2
   - 支持笔记: [[笔记3]], [[笔记4]]
   - 关键洞察: ...

## 矛盾/张力

- 想法冲突的地方
- 未解决的问题

## 差距

- 缺少什么
- 接下来要研究什么

## 联系

- 相关主题: [[主题1]], [[主题2]]
- 令人惊讶的联系: ...

## 建议的下一步

1. 需要的具体研究
2. 要探索的问题
3. 要尝试的实验
```

## 提示

- 最初广泛撒网,然后聚焦
- 寻找令人惊讶的联系
- 不要忽视矛盾 - 它们往往是洞察所在的地方
- 始终建议具体的下一步行动
