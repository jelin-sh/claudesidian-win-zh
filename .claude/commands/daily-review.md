# 每日回顾

进行每日回顾以捕获进展并为明天做准备。

## 回顾流程

1. **今天的活动**
   - 查找今天修改的所有笔记
   - 识别创建的新笔记
   - 审查所有项目的工作

2. **进展评估**
   - 完成了什么?
   - 什么被卡住或阻塞了?
   - 出现了什么意外的发现?

3. **捕获洞察**
   - 今天的关键学习
   - 发现的新联系
   - 出现的问题

4. **明天的设置**
   - 前 3 个优先事项
   - 要关闭的未闭环
   - 要探索的问题

## 输出格式

创建或更新每日笔记,包含:

```markdown
# 每日回顾 - [日期]

## 已完成

- ✓ [已完成项目 1]
- ✓ [已完成项目 2]

## 取得的进展

- [项目/领域]: [什么进展了]
- [项目/领域]: [什么进展了]

## 洞察

- [关键认识或联系]
- [重要学习]

## 被阻塞/卡住

- [什么没有进展以及为什么]

## 发现的问题

- [出现的新问题]
- [要研究的事情]

## 明天的重点

1. [优先事项 1]
2. [优先事项 2]
3. [优先事项 3]

## 未闭环

- [ ] [要记住的事情]
- [ ] [要跟进的人]
- [ ] [要发展的想法]
```

## 附加操作

- 将已完成的项目任务移至归档
- 更新项目状态笔记
- 链接相关的发现
- 标记需要注意的项目
